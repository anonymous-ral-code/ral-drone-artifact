 1) Check radio backend
python -c "import usb.backend.libusb1 as b; print(b.get_backend())"

Good output: not None.

2) Check Crazyradio USB
python -c "import usb.core; print(usb.core.find(idVendor=0x1915, idProduct=0x7777))"

Good output: not None.

3) Turn ON Crazyflie, then scan
python -c "import cflib.crtp; cflib.crtp.init_drivers(); print(cflib.crtp.scan_interfaces())"

Good output:

[['radio://0/80/2M', '']]

4) Test connection only
python -c "import cflib.crtp; from cflib.crazyflie.syncCrazyflie import SyncCrazyflie; cflib.crtp.init_drivers(); URI='radio://0/80/2M'; print('connecting'); scf=SyncCrazyflie(URI); scf.open_link(); print('connected'); scf.close_link()"
